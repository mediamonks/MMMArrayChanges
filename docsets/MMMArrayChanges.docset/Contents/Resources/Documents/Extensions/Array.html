Array

  • diffUpdate(elementId:sourceArray:sourceElementId:transform:update:remove:)

    Updates the receiver so it consists only of transformed elements from another array, avoiding transformations for the elements that are already in the receiver.

    This is similar to updating an existing array using map():

    array = sourceArray.map { /* transformation
  • compactDiffUpdate(elementId:sourceArray:sourceElementId:transform:update:remove:)

    The same as diffUpdate(elementId:sourceArray:transform:update:remove) except that it behaves as a Array.compactMap instead of Array.map, for cases where your source array can contain incomplete objects that might be populated in a future call.
